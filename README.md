SmartHydrate

SmartHydrate is a web-based application that helps users track daily water intake and stay properly hydrated. The app provides visual feedback, personalized recommendations, and weekly summaries to encourage healthy hydration habits.

Features
Home Page
Eye-catching hero section with hydration message.
Daily recommended water intake highlighted.
Easy navigation to the tracker page.
Water Intake Tracker
Input user name and age to personalize recommendations.
Track daily water consumption visually with interactive glasses.
View weekly progress, average intake, and completion rate.
Save progress or reset the weekly tracker.
Water intake recommendations by age displayed in a table.
Responsive Design
Works on desktop and mobile screens.
Uses CSS for interactive elements and progress bars.
Technologies Used
HTML5 – structure of the web pages.

CSS3 – styling, responsive layout, and animations.

JavaScript – dynamic water intake tracking and progress calculation.

File Structure
/tracker
  ├─ water.html       # Home page
  ├─ tracker.html     # Tracker page
  ├─ water.css        # Styles for all pages
  ├─ water.js         # JavaScript for tracker functionality

How to Use

Open water.html in a browser to see the home page.

Click Start Tracking or navigate to tracker.html.

Enter your name and age to get a recommended water intake.

Click on the water glasses to track daily intake.

View your progress and weekly summary.

Save or reset your progress using the buttons.

Notes

The tracker updates dynamically using JavaScript.

CSS animations give visual feedback when glasses are checked.

The app is mobile-friendly and adapts to smaller screens.
